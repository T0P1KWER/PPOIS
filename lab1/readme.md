 Lab 1: Менеджер фитнес-программ

> Объектно-ориентированная система управления тренировочными программами

Цель
Реализация классов для хранения, фильтрации и оценки фитнес-программ с сохранением данных в JSON.

  Основные классы

| Класс | Описание |
|-------|----------|
| `Fitness_program` | Модель программы: имя, уровень, длительность, упражнения, рейтинг |
| `ProgramsManager` | Управление коллекцией программ: загрузка, сохранение, поиск, рекомендации |
| `User` / `Trainer` | Модели пользователей и тренеров (расширяемость) |
